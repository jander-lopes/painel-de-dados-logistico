## Dashboard Financeiro

Dashboard para análise de dados de receitas, custos, resultado e margem de uma companhia logística.

### Linguagem de programação:

- python 3.10

### Gerenciador de dependências:

- poetry

### Bibliotecas:

- pandas
- numpy
- pathlib
- python-dotenv
- streamlit
- streamlit-autorefresh

### Controle de versões:

- git
